# Market Data Analyzer

A Python tool for analyzing financial market datasets.

This project demonstrates basic data analysis and visualization using Python.

## Features

- Load CSV market data
- Perform statistical analysis
- Calculate daily returns
- Visualize trends and trading volume

## Technologies

- Python
- pandas
- matplotlib
- seaborn

## Project Structure

market-data-analyzer
│
├── data
├── src
├── requirements.txt
└── README.md

## Installation

pip install -r requirements.txt

## Run

python src/main.py
